# 1.创建工程

```js
npx create-strapi@latest my-strapi-project
cd my-strapi-project && npm run develop
```

## 1.1 官网文档地址

https://docs.strapi.io/cms/quick-start

## 1.2 更改默认语言

```js
mv app.example.tsx app.tsx

打开 'zh-Hans',, 注释
```
